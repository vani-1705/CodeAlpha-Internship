# 🧮 Calculator — Web Application
A responsive calculator web application created using HTML, CSS, and JavaScript that performs basic arithmetic operations with a modern user interface and keyboard support.

---
# 📌 Project Overview
Calculator is a modern static web application designed to perform daily mathematical calculations efficiently through an attractive and user-friendly interface.
This project was developed as part of a Frontend Development Internship project using HTML, CSS, and JavaScript.

---
# ✨ Features
Responsive calculator design
Modern calculator UI
Basic arithmetic operations
Percentage calculation
Positive/Negative toggle
Decimal number support
Keyboard input support
Error handling for division by zero
Dynamic display resizing
Clean and readable interface

---
# 🔢 Operations Supported
Addition (+)
