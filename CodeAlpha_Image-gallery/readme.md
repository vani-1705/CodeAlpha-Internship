# 🖼 Image Gallery — Beautiful Moments

A modern and responsive Image Gallery website created using HTML, CSS, and JavaScript featuring category filters, image search, hover effects, lightbox view, image filters, dark/light mode, and responsive masonry layout.

# 📌 Project Overview

Beautiful Moments is a stylish and interactive image gallery project designed to showcase different categories of images including arts, animals, birds, cartoons, food, nature, temples, people, and more.

The gallery includes smooth animations, hover effects, image filtering options, search functionality, lightbox image preview, thumbnail navigation, dark/light theme toggle, and responsive layout for all devices.

This project was developed as part of a Frontend Development Internship task using HTML, CSS, and JavaScript.

# 🚀 Functionalities Implemented

## 🔍 Search Functionality
Users can search images using image titles or tags.

## 🏷 Category Filtering
Images can be filtered category-wise using category buttons.

## 🖼 Lightbox View
Clicking an image opens a fullscreen popup preview with navigation controls.

## ⬅➡ Navigation Buttons
Users can move between images using previous and next buttons.

## 🎨 Image Filters
Different visual filters can be applied dynamically.

## 🌙 Dark / Light Mode
Users can switch between dark and light themes.

## 📱 Responsive Design
The layout adapts smoothly across desktop, tablet, and mobile devices.
